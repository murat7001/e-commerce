# E-Commerce-App With React
## Start
- cd client
- npm install
- npm start
+ install redis and mongodb
+ cd backend
+ npm run dev

## Tools:
### Front
* ReactJS
* React Query
* Mockapi.io
* Chakra-UI
* Ant Design
* React Router
* Context API
* Formik
* Axios
### Backend
* Nodejs
* Expressjs
* Redis
* Mongodb
* jwt

## Features
- User and Admin Register & Login
- Delete, update and add new product
- Order
- Responsive Design
